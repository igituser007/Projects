# 📊 Google Data Trends Analysis

This repository contains a Jupyter Notebook for analyzing search trends using the **Google Trends** data via the [**Pytrends**](https://github.com/GeneralMills/pytrends) library (an unofficial API for Google Trends).

---

## 🔍 Project Overview

Using this notebook, you can:
- Analyze the popularity of keywords over time.
- Visualize search trends using `matplotlib`.
- Customize location (`geo`), category, timeframe, and keyword(s).
- Extend analysis with related queries or regional trends.
